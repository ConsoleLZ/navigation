## WV
WV,这个名字结合了 "Web"（网络）和 "Voyage"（航行、旅行）, 两个词是一个本人在学习过程中遇到的各种有用的网站与项目,包括一些软件, 于是便自己建立了一个导航网站方便自己查看,使用的是完全静态的框架[nuxt](https://nuxt.com.cn/),大家感兴趣的可以自己去看看😀

[demo](https://a-aa.fun/)

## 使用
#### 启动
```sh
npm run dev
```

#### 打包
```sh
npm run generate
```

#### 数据来源
所有的数据都在public/data目录下,可以自行配置
